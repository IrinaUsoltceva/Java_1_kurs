Объектно-ориентированное программирование

Программа соответствует предметной области,
объекты - "существительные" из предметной области
в программе поддержки обучения будут объекты
- студент
- преподаватель
- предмет
- экзамен
- оценка

объект содержит 1) данные и 2) поведение
1) студент: фио, курс, информация об оценках
2) преподаватель: фио, зарплата. предметы, которые ведет
3) оценка: значение, дата
...

Поведение:
1) Студент может получить оценку за предмет
2) студент может распечатать список всех своих оценок
3) преподаватель может получить новый предмет для обучения
...

В теории всё так, как написано выше. На практике, в реальных
программах примеры объектов обычно значительно более абстрактны

String - строки
Integer - число целое
File - путь на диске
Set - множество объектов
Scanner - устройство чтения слов из текста

DBConnection - связь с БД (базой данных)
CrawlerConfigFactory - фабрика конфигураций кроулера..

Мы будем программировать класс "рациональное число" Rational.
3/4, 8/7, 2 = 2/1, -5 = -5/1
